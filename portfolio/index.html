Classification de poissons avec Deep Learning

Projet de deep learning visant à distinguer deux espèces de poissons : Fundulus heteroclitus et Fundulus diaphanus. J’ai créé entièrement le dataset moi-même (collecte, nettoyage et annotation), puis entraîné un modèle de classification d’images avec Python, Keras et TensorFlow. Le modèle atteint une précision élevée et démontre ma capacité à gérer un pipeline complet d’apprentissage profond.

Python TensorFlow Keras Deep Learning Dataset personnalisé
